🔥 Firebase Configuration

Create a free project at console.firebase.google.com → Project Settings → General → Your Apps → Add Web App → copy the config object values below. Enable Authentication (Google provider) and Firestore Database in your project.

✨ Gemini API Key (optional — already configured)

A default Gemini key is built in. You can add your own key from aistudio.google.com if you prefer. Remember to restrict your key to your domain in Google Cloud Console.
